# 🛠️ سامانه مدیریت تعمیرگاه Samtronic

نرم‌افزار دسکتاپی برای مدیریت مشتری‌ها، ثبت تعمیرات، موجودی انبار و گزارش‌گیری در تعمیرگاه‌ها، با طراحی فارسی و رابط کاربری ساده.

## 💡 امکانات کلیدی

- 📋 مدیریت اطلاعات مشتریان
- 🛠️ ثبت و پیگیری سوابق تعمیرات
- 📦 مدیریت موجودی قطعات و کالاها
- 📊 تولید گزارش‌های ساده برای چاپ یا نگهداری
- 🎨 رابط گرافیکی با فونت IRANSans و آیکون اختصاصی
- 🐍 توسعه‌یافته با Python و PyQt5
